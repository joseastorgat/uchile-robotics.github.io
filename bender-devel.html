Bender is developed under Ubuntu 14.04LTS using the ROS Indigo framework. Manipulation and Navigation capabilities are built on top of standard libraries such as MoveIt and the ROS Navigation stack. Speech Recognition is achieved using Pocketphinx. Computer Vision is built on top of PCL and OpenCV, and we run our simulations on Gazebo and Stage.

Our code is hosted in public repositorieson on GitHub.
